инструкция по установке
1. скопировать папку ShopRSS в папку modules установленной hostcms
2. добавить в файл main_classes.php перед закрывающим тегом ?> строчку $GLOBALS['HOSTCMS_CLASS']['shoprss'] = 'ShopRSS';
3. установить модуль в админке выбрав при установке статус модуля -  активен (при этом создадутся необходимые типовые динамические страницы)

Использование модуля
1. Инсталировать модуль
2. Создать узел структуры для трансляции RSS:
тип - типовая динамическая страница;
раздел типовых динамических страниц - RSS для интернет-магазина
типовая динамическая страница - RSS для интернет-магазина
после выбора типовой линамической страницы появится еще несколько полей:
Интернет-магазин - выберите интернет-магазин с которого нужно организовать трансляцию
Число выводимых элементов в ленте - по умолчанию задано 15
Заголовок RSS-канала - если не заполнено, то в качестве названия используется название интернет-магазина 
Описание RSS-канала - если не заполнено, то в качестве названия используется описание интернет-магазина 
Удалять теги из RSS - включает очистку тегов в элементах RSS трансляции
Отображать изображение для товара в RSS - включить отображение изображений для товаров
Включать фото для товара в текст элемента - включает вставку изображения для товара в текст элемента, в противном случае оно прикрепляется к элементу как вложение
